# Klinik_FrontDesk
**Klinik Bagian Admin / Front Desk**
Programming Language  : Python (Backend), Dart (Frontend)
Script                : Python
Framework             : FastAPI (Backend), Flutter (Frontend Web)
UI Library            : Flutter Material
Styling               : Custom Glassmorphism (via BackdropFilter & ImageFilter), Gradient Backgrounds
Styling Utils         : dart:ui, standard Flutter BoxDecoration
Database              : MySQL  
ORM                   : SQLAlchemy
Validation            : Pydantic (Backend)
Authentication        : JWT (JSON Web Tokens) dengan bcrypt Hashing (OAuth2PasswordBearer)
UI Components         : Custom GlassContainer, NavigationRail, Material Widgets
Icons                 : Material Icons (Flutter Default)
File Processing       : python-dotenv (Environment Variables)
Testing (E2E)         : Postman (Manual Collection provided)
Testing Framework     : unittest (Python Standard), flutter test
Integration           : Frappe / ERPNext (REST API Sinkronisasi Pasien & Antrian)
